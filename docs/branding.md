# Style Guide : Mina Glamour

Ce document définit l'identité visuelle de la marque pour assurer une cohérence sur tous les supports (Site Web, Réseaux Sociaux, Packaging).

## 1. Palette de Couleurs (Design Tokens)

| Token | Rôle | Valeur Hex |
| :--- | :--- | :--- |
| `--color-primary` | Fond Principal / Texte Clair | `#FDF8F0` (Champagne) |
| `--color-secondary` | Texte Principal / Profondeur | `#1A1A1A` (Noir Onyx) |
| `--color-accent` | Boutons / Détails Luxe | `#D4AF37` (Or Mat) |
| `--color-glamour` | Alertes / Éclat | `#043927` (Émeraude) |

## 2. Typographie

- **Titres (Headings)** : *Cormorant Garamond* (ou Serif luxueux similaire).
  - Utilisation : H1, H2, H3.
  - Sentiment : Héritage, Élégance, Prestige.
- **Corps de texte (Body)** : *Montserrat* ou *Inter* (Sans-Serif moderne).
  - Utilisation : Paragraphes, Boutons, Menus.
  - Sentiment : Clarté, Modernité, Accessibilité.

## 3. Logo & Image
- **Logo Principal** : Doit être utilisé sur fond Champagne ou Noir Onyx.
- **Iconographie** : Lignes fines, minimalistes, style "Line Art".

## 4. Ton & Voix
- **Mots clés** : Reine, Éclat, Confiance, Souplesse, Naturel.
- **Slogan** : *Cheveux de rêve, allure de Reine.*
