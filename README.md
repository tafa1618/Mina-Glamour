# Mina Glamour

Boutique e-commerce premium au Sénégal spécialisée dans les cheveux naturels et la mode féminine.

## Vision
**Mina Glamour : Cheveux de rêve, allure de Reine.**
Marque féminine, élégante et moderne avec une forte identité africaine.

## Architecture Technique
- **CMS Headless** : WordPress + WooCommerce (WPGraphQL)
- **Frontend** : Next.js (App Router)
- **Paiements** : Wave & Orange Money
- **IA** : Système multi-agents (Trends, Sourcing, Marketing/SEO)

## Structure du Projet
- `/frontend` : Application Next.js
- `/backend` : Configuration WordPress / Docker
- `/docs` : Documentation stratégique et design
